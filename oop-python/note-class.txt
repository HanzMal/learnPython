Cara Dinamis: Kamu diberi selembar kertas oleh user yang bertuliskan nama lacinya (misal user menulis "surat_tanah" atau "berlian"). Kamu tidak tahu apa yang ditulis user sampai kertas itu dibuka. Di sini, kamu butuh alat (getattr) untuk membaca tulisan di kertas itu, lalu membuka laci yang sesuai secara otomatis.

Bayangkan kamu sedang membuat sistem Odoo atau aplikasi Profile Pengguna. Kamu membuat Class kosong seperti ini:
class Karyawan:
    pass

budi = Karyawan()

kalau cuman 1 underscore hanya menunjukkan sebuah peringatan/kesepakatan (konvensi) sosial namun tetap bisa diakses dari luar class
double underscore di method class python itu untuk menunjukkan bahwa akses method tersebut private dan jangan mengaksesnya dari luar Class

Class Atribute = Atribut yang didefinisikan langsung di dalam badan class (di luar method apa pun). Nilai dari atribut ini dibagi bersama (shared) oleh semua objek/instance yang dibuat dari kelas tersebut. Jika nilainya diubah melalui kelas, semua objek akan ikut berubah.

Instance attribute(Atribut Objek) = Atribut yang didefinisikan di dalam method (biasanya di dalam konstruktor __init__) menggunakan kata kunci self. Atribut ini bersifat unik untuk setiap objek. Perubahan pada satu objek tidak akan memengaruhi objek lainnya.
